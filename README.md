## **Project Overview**

**Image Rotation and Resize** is a digital image processing project that focuses on performing basic geometric transformations on images. The project demonstrates how images can be rotated by a specified angle and resized to different dimensions while maintaining image quality.

---

## **Objectives**

* **To understand basic image transformations**
* **To implement image rotation**
* **To perform image resizing**
* **To analyze the effect of transformations on image quality**

---

## **Features**

* **Rotate images by any angle**
* **Resize images to custom dimensions**
* **Support for common image formats**
* **Simple and efficient implementation**

---

## **Tools and Technologies Used**

* **Programming Language:** *e.g., Python / MATLAB / C++*
* **Libraries:** *e.g., OpenCV, NumPy, PIL*
* **Platform:** *e.g., Windows / Linux*

## **How It Works**

1. **Input an image**
2. **Apply rotation using a specified angle**
3. **Resize the image to desired width and height**
4. **Display and save the processed image**
